# 🌐 Company Profile

A modern, responsive company profile website built with **React**, **TypeScript**, and **Vite**. Designed with a clean user interface, smooth user experience, and reusable component architecture. The project features responsive layouts, dark mode support, interactive UI components, and robust form validation.

## ⚙️ Getting Started

### Clone Repository

```bash
git clone https://github.com/Hary300/Personal-Project-27-Company-Profile.git
```

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```
